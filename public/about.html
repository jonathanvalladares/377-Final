About CyberFeed

Learn more about this project and why it was built.

What is CyberFeed?

CyberFeed is a Cybersecurity News and Threat Feed Dashboard that brings together the latest cybersecurity news from across the web into one clean, organized interface. Instead of having to visit multiple websites to stay informed about threats like ransomware, phishing, data breaches, and malware, you can find it all here.

Who is this for?

This dashboard is built for:

  • Cybersecurity students learning about the current threat landscape
  • IT administrators who need to stay on top of emerging threats
  • Developers who want to be aware of security vulnerabilities
  • Small business owners who care about protecting their data
  • Anyone interested in cybersecurity and digital safety

Features

  • Search for any cybersecurity topic using keyword search
  • Filter articles by threat category (Ransomware, Phishing, Malware, etc.)
  • View article summaries with source, date, and description
  • Click "Read More" to view the full article on the original source
  • Save or bookmark articles to read later on the Saved Articles page
  • Visual chart showing the breakdown of threat types in current results

Technology Stack

This project was built using the following technologies:

HTML CSS JavaScript Node.js Express Supabase GNews API Chart.js Day.js

Why GNews API?

The GNews API was chosen because it supports keyword-based searches and returns well-structured data including article titles, descriptions, publication dates, source names, and images. This makes it easy to pull in relevant cybersecurity news and present it in a clean, readable format.

Data Storage

Supabase is used as the backend database to store bookmarked articles. When you save an article on the home page, it gets stored in a Supabase database and can be retrieved on the Saved Articles page at any time. You can also delete saved articles when you no longer need them.

About This Project

This application was built as a final project for INST 377 at the University of Maryland. The goal was to design and build a full-stack web application that solves a real problem using modern web technologies including a Node.js backend, external API integration, and a Supabase database.
