About CyberFeed

Learn more about this project and why it was built.

What is CyberFeed?

CyberFeed is a cybersecurity news dashboard that pulls live articles from around the web and puts them all in one place. Instead of going to a bunch of different sites to find news about hacking, ransomware, or data breaches, you can just search for it here.

Who is this for?

This site is useful for::

  • Students learning about cybersecurity
  • IT people who want to stay updated on threats
  • Developers who care about security
  • Anyone interested in cybersecurity and digital safety

Features

  • Search any cybersecurity topic
  • Filter by threat type like ransomware, phishing, or malware
  • View article summaries with source, date, and description
  • See where articles are from and when they were published
  • Save articles to read later
  • Visual chart showing the breakdown of threat types in current results

Technologies

This project was built using the following technologies:

HTML CSS JavaScript Node.js Express Supabase GNews API Chart.js Day.js

Why GNews API?

GNews lets you search for news articles by keyword and gives back useful info like the title, description, source, and image. It was pretty easy to work with and fit exactly what this project needed.

Data Storage

Saved articles get stored in a Supabase database. When you click save on an article it gets added to the database and shows up on the Saved Articles page. You can also delete them from there.
